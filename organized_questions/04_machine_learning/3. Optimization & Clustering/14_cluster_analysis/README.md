# Cluster Analysis Interview Questions

This folder contains comprehensive interview questions about cluster analysis for machine learning interviews.

## Question Types Available:

- **Theory**: 50 questions (K-Means, DBSCAN, Hierarchical, GMM, BIRCH, OPTICS, etc.)
- **Scenario_Based**: 11 questions (Customer segmentation, Image segmentation, Time-series clustering, etc.)
- **General**: 5 questions (Preprocessing, GMM, Elbow method, etc.)
- **Coding**: 8 questions (Python implementations with sklearn)

**Total Questions**: 74

## Key Topics Covered:

### Algorithms
- K-Means (initialization, limitations, K-means++)
- DBSCAN (core/border/noise points, eps, MinPts, k-distance plot)
- Hierarchical (Agglomerative vs Divisive, linkage methods)
- GMM (soft clustering, EM algorithm)
- BIRCH, OPTICS, HDBSCAN, Mean Shift, Spectral Clustering

### Concepts
- Distance metrics (Euclidean, Manhattan, Cosine, Haversine)
- Cluster validation (Silhouette, Davies-Bouldin, Calinski-Harabasz)
- High-dimensional clustering (curse of dimensionality, subspace clustering)
- Scaling and normalization importance

### Applications
- Customer segmentation (RFM analysis)
- Image segmentation
- Document/topic clustering
- Geographic clustering
- Social network community detection
- Time-series clustering
