# 100 Days of Deep Learning — Course Announcement

## Course Structure (5 Sections)

| Section | Topic | Estimated Days |
|---|---|---|
| 1 | Artificial Neural Networks (ANN) | ~30–35 days |
| 2 | Convolutional Neural Networks (CNN) | ~20 days |
| 3 | Recurrent Neural Networks (RNN) | ~20 days |
| 4 | Generative Models (GANs, Autoencoders) | ~15 days |
| 5 | Advanced Topics (Object Detection, Image Segmentation) | ~10 days |

---

## Section 1: ANN (Foundation) — Detailed Curriculum

### Basics
- What is Deep Learning
- Deep Learning vs Machine Learning
- Applications and types of Deep Learning
- History of Deep Learning

### Perceptron
- What is a Perceptron (building block of neural networks)
- Relationship with biological neurons
- Prediction in a perceptron
- Training a perceptron
- Limitations → need for Multi-Layer Perceptron (MLP)

### Multi-Layer Perceptron / ANN
- MLP architecture and intuition
- Forward propagation (prediction)
- **Backpropagation** (most important topic)
- Gradient Descent

### Practical
- Installation of deep learning libraries (TensorFlow/Keras)
- Regression example project
- Classification example project

### Improving Neural Networks
- Activation functions (sigmoid, tanh, ReLU)
- Dropout layers
- Regularization techniques
- Optimizers (SGD, Adam, etc.)
- Batch Normalization
- Hyperparameter tuning

### Advanced ANN Topics
- Callbacks
- TensorBoard visualization
- Transfer Learning (using pre-trained models)
- Keras Functional API (vs Sequential)
- Model saving and loading
- Streamlit deployment
- **End-to-end project** (full application)
